# 복지한입 — 흩어진 복지를 한 입에

> 2026 AI 챔피언 해커톤 · 본선 부문1 "복지·혜택"
> 시제: **흩어진 복지·혜택을 국민에게 닿게 하라**

로그인 없이 **말 한 줄(또는 글)**로 내 상황을 말하면, 받을 수 있는 복지를
**"왜 되는지" 근거와 함께 쉬운말·음성**으로 풀어주고, **신청 준비까지** 한 흐름으로 안내합니다.

## 핵심 차별점
- **무로그인·무동의** — 개인정보 연계 없이 자가서술만으로 즉시
- **이해** — 행정용어 줄글을 쉬운말 + 음성(TTS)으로
- **근거** — 가능/불확실/추가확인 3분류 + "왜 되는지" 선정기준 인용
- **원스톱** — 발견 → 이해 → 자격 가능성 → 신청준비 → 연결
- **접근성** — 음성 입력(STT)·큰 글씨·고대비

> 기존 정부 서비스(복지로·복지멤버십)는 실데이터로 '발견'을 잘하지만 로그인·동의의 벽이 있고
> "이해"는 돕지 않습니다. 우리는 그 **입구**를 맡습니다. (발견 정확도 경쟁 X)

## 기술 스택
- Next.js (App Router) + TypeScript + Tailwind
- **Claude Opus 4.8** (에이전트 사실추출·공감문구) — `/api/agent`
- **OpenAI** Whisper(STT) `/api/stt` · TTS `/api/tts`
- 매칭·자격판정은 **결정론 규칙(JS)** → 키 없이도 작동 (LLM/음성은 폴백 체인)

## 데이터
- 한국사회보장정보원 **중앙부처복지서비스** (data.go.kr 15090532) 기반 큐레이션
- 실직 생애사건 6개 복지 (`src/data/welfare.ts`)
- ⚠️ 자격 수치는 데모용 근사치 — 운영 시 공식 고시값으로 대체 필요

## 실행
```bash
npm install
cp .env.local.example .env.local   # 키 입력(선택 — 없어도 동작)
npm run dev                         # http://localhost:3000
```
키가 없어도 **결정론 매칭 + 사전 쉬운말 + 브라우저 음성**으로 전체 흐름이 작동합니다.

## 폴백 체인 (데모 안정)
- Claude 실패 → 키워드 휴리스틱 + 사전 캐싱 쉬운말
- OpenAI STT/TTS 실패 → 브라우저 Web Speech → 타이핑/텍스트

## 구조
```
src/
  app/
    page.tsx          # 입력→프록시→결과→상세 4단계 플로우
    api/agent|stt|tts # Claude · OpenAI 라우트
  data/welfare.ts     # 복지 데이터(중앙부처복지서비스 기반)
  lib/
    match.ts          # 검색+자격판정 3분류 (결정론)
    income.ts         # 기준중위소득 러프 스크린
    extract.ts        # 휴리스틱 사실추출(LLM 폴백)
    voice.ts          # STT/TTS 클라이언트(폴백 포함)
```
